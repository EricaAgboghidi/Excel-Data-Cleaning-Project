# Data-Cleaning-Project
## Project Overview
This project focuses on preparing a raw dataset for analysis by improving its quality, consistency, and usability. The objective was to identify and resolve common data quality issues, ensuring the dataset was clean and ready for further analysis.
Objectives
- Identify and handle missing or null values. 
- Detect and remove duplicate records. 
- Correct incorrect data formats. 
- Improve overall data consistency and integrity.

## Data Cleaning Process
The following data cleaning tasks were performed:
1. Duplicate Check:  
Inspected the dataset for duplicate records. 
No duplicate entries were found. 
2. Date Formatting: 
Converted the Date column from Text to Short Date format to enable accurate date-based analysis. 
3. Currency Formatting: 
Converted the Unit Price and Total Price columns from Text to Currency format to support financial calculations. 
4. Numeric Data Formatting: 
Converted the Quantity and Items in Cart columns from Text to Whole Number format for accurate numerical analysis. 
Handling Missing Values 
5. Replaced blank cells in the CouponCode column with "N/A" to standardize missing values and improve data consistency.

## Data Quality Outcome
After the cleaning process:
- No duplicate records remained. 
- Missing values in the CouponCode column were standardized. 
- Date fields were converted to the correct date format. 
- Numeric and currency fields were assigned appropriate data types. 
- The dataset was clean, consistent, and ready for analysis. 

## Conclusion
This project demonstrates the importance of data cleaning as a foundational step in the data analysis process. By addressing missing values, validating data types, and checking for duplicate records, the dataset was transformed into a clean and reliable resource for analysis. Although the cleaning tasks were straightforward, they highlight essential data preparation skills that help ensure accurate insights and support informed decision-making
